# 🤖 Autonomous Mode Guide

Understanding how the autonomous system works and how to control it.

---

## Overview

The Autonomous Orchestrator is the **brain** of the system. It runs daily (3:00 AM) and makes decisions automatically:

- 📊 Analyzes performance
- 📈 Scales successful agents
- ⚡ Optimizes pricing and strategies
- 🔍 Discovers new niches
- 💰 Withdraws profits
- 🌐 Evaluates new marketplaces

**You only receive reports and approval requests in Telegram.**

---

## Configuration

### Enable/Disable Autonomous Mode

Edit `config/autonomous.json`:

```json
{
  "enabled": true  // Set to false to disable
}
