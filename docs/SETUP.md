# 🚀 Setup Guide

Detailed step-by-step setup instructions for CashClaw Autonomous Farm.

---

## Prerequisites

### Required

- **Node.js** >= 18.0.0
- **npm** >= 9.0.0
- **Git**
- **OpenRouter API key** ([get here](https://openrouter.ai))
- **Telegram bot** ([create with BotFather](https://t.me/BotFather))

### Recommended

- **VPS** (for 24/7 operation)
  - Aeza START-1 ($3.5/month)
  - Hetzner CX11 (€4.5/month)
  - DigitalOcean Basic ($6/month)
- **SSH access** to VPS
- **Base network ETH** (for gas fees)
